# Telco_Customer_churn_Analysis
Customer Churn Analysis Project using Microsoft Excel and Power BI to identify customer retention patterns, churn trends, and business insights. Performed data cleaning, KPI Analysis, inteeractive dashboard creation, and visualization to support data-driven decision-making and improve customer retention strategies.
# Files Included
Telco_Customer_Churn_Dataset.xlsx - Customer Data
Telco_Customer_Churn_Dataset.pbix - Power BI Dashboard
# Dashboard Highlights
- Churned Customer
- Churn Rate
- Sales Distribution
- Contract Distribution
- Churn by Tenure Group
- Churn by Monthly Charges
- Churn by Contract, Payment Methods and Tech Support
# Tools Used
- Microsoft Excel
- Power BI
